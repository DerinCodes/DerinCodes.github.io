Perceptron for AND Logic Gate

This project implements a single-layer perceptron to classify binary inputs as either 0 or 1, following the logic of an AND gate. The perceptron is trained using a basic dataset and adjusted using gradient descent.

Code Breakdown

1. Perceptron Class Definition

The Perceptron class initializes with a given input size, a learning rate, and uses a step function for binary classification.

class Perceptron:
  def __init__(self, input_size, learning_rate=0.1):
    self.weights = np.zeros(input_size)
    self.bias = 0
    self.learning_rate = learning_rate

2. Predict Method

The predict method uses the step activation function to output either 0 or 1. The activation is calculated as (inputs • weights) + bias, and the output is 1 if the result is greater than or equal to 0, else 0.

def predict(self, inputs):
  return np.where((np.dot(inputs, self.weights) + self.bias) >= 0, 1, 0)

3. Fit Method

The fit method trains the perceptron over multiple epochs by updating weights and bias based on prediction errors. The weights and bias are adjusted in the direction that reduces the error.

def fit(self, X, y, epochs=100):
  for _ in range(epochs):
    for inputs, target in zip(X, y):
      prediction = self.predict(inputs)
      self.weights += self.learning_rate * (target - prediction) * inputs
      self.bias += self.learning_rate * (target - prediction)

4. Training the Perceptron

The perceptron is trained on a dataset representing the AND logic gate, where the expected outputs are 0 unless both inputs are 1.

X_train = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y_train = np.array([0, 0, 0, 1])

perceptron = Perceptron(input_size=2)
perceptron.fit(X_train, y_train)

5. Testing and Predictions

The perceptron’s predictions for the AND gate inputs are displayed, showing whether the model correctly learned the logic gate’s behavior.

test_inputs = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
predictions = perceptron.predict(test_inputs)
for inputs, prediction in zip(test_inputs, predictions):
  print(f"Inputs: {inputs}, Prediction: {prediction}")
